# Food Delivery Performance Analysis

## Project Overview
This project analyzes food delivery data using Python and Pandas. It provides insights into delivery performance based on traffic, weather, city, vehicle type and delivery time.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Dataset Information
- Delivery Time Distribution
- Average Delivery Time by City
- Average Delivery Time by Weather
- Average Delivery Time by Traffic
- Average Delivery Time by Vehicle Type
- Delivery Count by City
- Ratings Distribution
